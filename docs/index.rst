Easy-LIN Documentation
======================

Easy-LIN is a Python application for LIN engineering, built with PySide6.
The documentation below focuses first on technical scope: LDF parsing,
hierarchy inspection, LIN communication workflows, runtime behavior, and API
surfaces for version 0.8.2.

Planned releases:

- 0.7.3 for runtime hardening and diagnostics
- 0.8.0 for the next larger communication workflow increment

The communication panel is now a separate window (View > Communication
Window or Ctrl+Shift+C), allowing independent management of LDF analysis
and hardware communication.

Accessibility support is documented as a dedicated section in the overview
after the core technical workflow description.

RGAA 4.1.2 (automatable criteria scope): 100% compliant on implemented
automatic checks (16/16), validated by automated tests.
