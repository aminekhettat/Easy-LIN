Easy-LIN Documentation
======================

Easy-LIN is an accessibility-first Python application for LIN engineering. The
documentation below covers the public entry points, source modules, preserved
PyQt compatibility path, tooling, and automated tests for version 0.5.2.

RGAA 4.1.2 (automatable criteria scope): 100% compliant on implemented
automatic checks (16/16), validated by automated tests.

Known limitation for version 0.5.2: accessibility improvements are in place,
but the interface is still not reliably usable in day-to-day operation.
Additional stabilization is required before production usage.
